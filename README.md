# Job Portal System

A full-stack Job Portal web application built using ASP.NET Core MVC and Web API with JWT Authentication, role-based authorization, resume upload, employer dashboards, analytics, and responsive modern UI.

---

## Features

- JWT Authentication
- Role-Based Authorization
- Employer Dashboard
- Candidate Dashboard
- Job Posting
- Resume Upload
- Application Tracking
- Accept / Reject Applications
- Search & Filters
- Dark Mode UI
- Company Logo Upload
- Responsive Design

---

## Tech Stack

- ASP.NET Core MVC
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Bootstrap 5
- JWT Authentication
- Clean Architecture
